# Agenda de Contactos

### Descripción

Esta aplicación es una Agenda de Contactos construida con **_React.js_**, enfocada en el manejo del **estado**, la **inmutabilidad** y la **persistencia de datos** utilizando **_localStorage_**.

### Herramientas

- _CSS_
  - ❌ _Responsive_
- _JavaScript_
- _React_
  - _jsx_
  - _props_
  - Renderizado de Elementos (lista)
  - _useState (Hook)_
  - _useEffect (Hook)_

### Implementación

- _*CRUD*_
- Persistencia de Datos en **_localStorage_**
